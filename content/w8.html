在 Ubuntu 安裝 gd 程式庫套件:

sudo apt update

sudo apt upgrade

sudo apt install libgd-dev

cd downloads

cc gd_ex1.c -lm -lgd

./a.out

在 Ubuntu 安裝 gnuplot 套:

sudo apt install gnuplot

因為 s1511 伺服器並沒有安裝 X11, 因此 raylib 相關範例只能在近端可攜環境中執行.
